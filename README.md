# company-search app
A dashboard to view companies data

## Project setup - in client & server directories 
```
npm install
```

## Run scarper - `server/`
```
node scraper.js
```

## Run Server (http://172.19.181.123:3000/), in `server/` directory:
```
npm start
```

## Run Vue app (http://172.19.181.123:8080/) in `client/` directory:
```
npm run serve
```
